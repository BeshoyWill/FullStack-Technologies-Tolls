Steps that you need to begin with in every project

create folders for :-

  1. components.   ( layout / additions )
  2. core. ( interfaces / services / interceptors / environment / guards )

Begin With the data you know like handling services for example if you're working with apis.

 

To create these files using command line :-

ng g c ===> create component

ng g i ===> create interface

ng g s ===> create service

ng g interceptor ===> create interceptor

ng g g ===> create guard
