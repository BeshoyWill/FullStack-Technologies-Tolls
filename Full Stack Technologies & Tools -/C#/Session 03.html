# 2) Implicit Casting & Explicit Casting

Implicit Casting :-

--→ Implicit casting happens automatically when a smaller type is assigned to a larger type, and there is no risk of data los.

int x = 10;

long y = x;

Console.WriteLine(y);

Explicit Casting :-

--→ Explicit casting (manual casting) is needed when converting from a larger type to a smaller type, or from an incompatible type. This can result a data loss or errors.

long x = 12342134;

int y = (int)x

Console.WriteLine(y);

Unary Operators [ ++, -- ] :-

// 2. ++ ===> Postfix [Print Then Increment]
