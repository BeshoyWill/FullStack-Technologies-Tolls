Initialize a Git Repository

 

Initialize a Git Repository (Brief Explanation)

To start using Git in a project, you need to initialize a Git repository. This creates a .git directory in your project folder, which Git uses to store the project's history and track changes.

Steps to Initialize a Git Repository:

Open Terminal:
Navigate to your project folder using the terminal or command line.

Run the Git Init Command:
Inside the project directory, run the following command:

This initializes a new Git repository in your project by creating a hidden .git folder where Git will track all changes.

bash

Copy code

git init

Add Files to the Repository:
To start tracking files, add them to the staging area using:

Or add all files:

bash

Copy code

git add .

bash

Copy code

git add <filename>

Commit the Changes:
Save the changes to the repository with:

bash

Copy code

git commit -m "Initial commit"

Result:

After running git init, your project is now a Git repository, and Git will track any changes you make.

 

Explaining in Arabic :-

 

تهيئة مستودع Git (شرح مختصر)

لبدء استخدام Git في مشروعك، يجب عليك تهيئة مستودع Git. هذا يقوم بإنشاء مجلد مخفي يسمى .git داخل مجلد مشروعك، وهو المكان الذي يستخدمه Git لتخزين تاريخ المشروع وتتبع التغييرات.

خطوات تهيئة مستودع Git:

فتح الطرفية (Terminal):
انتقل إلى مجلد مشروعك باستخدام الطرفية أو سطر الأوامر.

تشغيل أمر Git Init:
داخل دليل المشروع، قم بتشغيل الأمر التالي:

#bash

git init

 

هذا يقوم بتهيئة مستودع Git جديد في مشروعك عن طريق إنشاء مجلد مخفي .git حيث سيقوم Git بتتبع جميع التغييرات.

إضافة الملفات إلى المستودع:
لبدء تتبع الملفات، أضفها إلى منطقة التحضير باستخدام:

#bash

git add <اسم_الملف>

 

أو لإضافة جميع الملفات:

#bash

git add .

 

الالتزام بالتغييرات:
لحفظ التغييرات في المستودع، استخدم الأمر التالي:

#bash

git commit -m "أول التزام"

النتيجة:

بعد تشغيل git init، يصبح مشروعك مستودعًا لـ Git، وسيبدأ Git في تتبع أي تغييرات تجريها على الملفات.
